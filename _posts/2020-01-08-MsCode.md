# 발열킹 노트북
세대가 올라 갈수록 코어와 발열이 수직 상승 하고 있어
얇게 나온 노트북이면 엄청 뜨겁거나 느리기 마련이다.
쉽게 100도 까지 찍어낸다.

무거운 작업을 하기 위해서 
집에서 운영하는 리눅스 서버에
포트 포워딩을 통해 주피터 노트북을 별도로 운용 하고 있었다.

파이썬만 사용 가능 했고
별도의 코딩은 좀 그랬다.

# MS CODE 의 확장성

온라인 코딩 테스트 처럼 이용 한다.
